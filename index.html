Інструкція користувача

Інструкція користувача до файлу «PI‑RADS — звіт»

Призначення

  • Інтерактивне відмічання секторів простати у сагітальній, коронарній та аксіальній проекціях.
  • Введення розмірів простати TR/AP/CC (мм) і автоматичний розрахунок об’єму.
  • Введення ПІБ пацієнта з відображенням у звіті.
  • Автоматичне формування текстового звіту PI‑RADS і друк.

Основні блоки

  • Параметри простати: TR, AP, CC (мм) та поле ПІБ під ними. ПІБ відображається першим рядком у звіті.
  • Інтерактивні зображення: кліки по секторах (база, мід, апекс) вибирають сегменти для звіту.
  • Звіт (праворуч): містить ПІБ, розміри/об’єм простати, вибрані сектори, оцінки.

Кнопки керування

  • Редагувати полігони: режим зміни контурів секторів (додати/перемістити/видалити точки).
  • Редагувати маркери: переміщення еліпсів уражень для ручної корекції.
  • Автопозиція маркерів: автоматично центрирує маркери в обраних секторах.

Налаштування маркерів

  • Кольори уражень: Колір 1/2/3 для візуального розрізнення на схемах.
  • Зміщення еліпсів: повзунки Sagittal X/Y та Coronal X/Y (від −100% до +100%). Кнопка «Скинути» повертає нулі.

Робота з полігонами

  • Увімкнути: натисни «Редагувати полігони».
  • Додати точку: клік лівою кнопкою по відрізку полігона.
  • Перемістити точку: перетягни маркер-вузол мишею.
  • Видалити точку: Alt + клік по точці.
  • Тимчасово прибрати під час редагування: Shift при перетягуванні.
  • Вихід: повторно натисни «Редагувати полігони».

Алгоритм користування

  1. Введи TR/AP/CC (мм) і ПІБ.
  2. Відміть потрібні сектори на проекціях.
  3. За потреби відкоригуй полігони/маркери, кольори та зміщення.
  4. Переглянь сформований звіт праворуч і роздрукуй.

📁 Вибір папки для збереження

  • Натисніть кнопку ☰ (праворуч угорі) → Обрати папку для збереження.
  • Після вибору всі подальші збереження (зображення/файли) йдуть у цю папку.
  • Якщо папку не обрано, файли автоматично зберігаються у стандартну теку «Завантаження» браузера.

🧾 Історія пацієнтів

  • Кнопка Історія у верхній панелі відкриває список збережених пацієнтів.
  • Зберегти поточного — зберігає ПІБ, усі поля, полігони (pirads21_*) та вміст звіту.
  • У списку: Відкрити — відновлює повний стан; Видалити — прибирає запис.
  • Після зміни ПІБ стан автоматично перезаписується під цим ім’ям.
